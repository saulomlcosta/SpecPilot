# ADR 0007 - Usar testes automatizados

## Status

Aceita

## Contexto

O projeto precisa demonstrar qualidade de engenharia e confianca de evolucao.

## Decisao

Adotar testes unitarios e de integracao desde as proximas etapas de implementacao.

## Consequencias

- maior seguranca para refatoracoes
- feedback rapido
- melhor demonstracao academica de qualidade
