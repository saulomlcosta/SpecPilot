# ADR 0007 - Usar testes automatizados

## Status

Aceita

## Contexto

O projeto precisa demonstrar qualidade de engenharia e confianca de evolucao.

## Decisao

Adotar testes unitarios e de integracao desde as proximas etapas de implementacao.

## Alternativas consideradas

- validacao apenas manual
- testes apenas de integracao

## Justificativa

Somente validacao manual enfraqueceria a confianca evolutiva do projeto. Somente testes de integracao deixariam regras de negocio importantes sem feedback rapido e isolado.

## Consequencias

- maior seguranca para refatoracoes
- feedback rapido
- melhor demonstracao academica de qualidade
